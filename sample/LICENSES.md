# Sample clips — licenses & attribution

These four clips are short, downscaled excerpts of **Blender Foundation open
movies**, bundled so a fresh arkiv can demonstrate search on first run without
your own footage. Each is licensed **Creative Commons Attribution (CC-BY)** and
is redistributed here with attribution, as the license requires. arkiv's own code
is MIT; these media assets keep their CC-BY license.
